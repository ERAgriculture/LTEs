# Contributing

This repository follows the **shared ERA contributing guidelines**:
<https://github.com/ERAgriculture/era-program/blob/main/CONTRIBUTING.md>

In short: open or find an issue, branch from `main`, keep changes small and
reviewable, add tests/docs, run checks, open a PR (a maintainer reviews; CI must
be green). Never commit secrets or bulk data. New repos are lowercase `era-*`.

Repo-specific notes live in this repo's `README.md` (and `CLAUDE.md` if present).
